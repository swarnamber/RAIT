This project mainly focuses on the basic but important topics in Selenium scripted with Java. 
Also there is a youtube recording link given for a live session conducted by me.
Very helpful in revising the entire selenium webdriver concept within small timeline
Very helpful for preparation for interviews.


Pre requisite
--------------
Understanding of core java
OOPS concept
Collection Framework


Takeaways
----------
After hands on this project audience will be able to -
  Get the feel and experience of automation testing.
  Automate any simple webpages 
  Experiment with other actions/methos
  Automate small daily tasks
  Clear basic interview questions
